❗ **중요: PDF 파일 순차 처리 필수**

- 동시에 2개 이상 PDF 읽기 금지
- 이유: API 413 에러 발생
- 해결: 챕터별로 1개씩 순차 처리

**올바른 방법:**
1. 챕터 1개 선택
2. 해당 PDF 1개만 읽기
3. 개선 완료
4. 다음 챕터로 이동

**잘못된 방법:**
- 여러 PDF 동시 읽기 ❌
- 병렬 처리 ❌

### 품질 기준

각 챕터 개선 시:
- 원본 PDF 내용 90% 이상 반영
- 01장과 동일한 구조 및 품질
- 목표 줄수 ±10% 이내
- 모든 체크리스트 항목 완성

### 일정 관리

각 챕터당 80분 초과 시:
- 진행 상황 점검
- 우선순위 재조정
- 필요시 사용자와 상의
