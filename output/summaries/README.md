## 📖 사용 방법

### 1. 빠른 복습 (10분 이내)
- **각 챕터의 "한 문장 요약"만** 읽기
- 전체 핵심을 빠르게 파악

### 2. 심화 학습 (30분 이내)
- **관심 챕터의 전체 요약** 읽기
- 핵심 개념 + 주요 교훈 + 실행 가이드 포함

### 3. 실무 적용
- **체크리스트 섹션** 활용
- 자신의 프로젝트에 맞춰 검증

### 4. 기초부터 체계적 학습
- 00 → 01 → 02 순서로 **순차 진행**
- 처음부터 끝까지 읽으면 성능 최적화의 전체 방법론 습득

## 🎯 요약 형식

각 요약은 다음 **5개 섹션**으로 구성됩니다:

### 📌 핵심 개념
- 챕터의 주요 개념과 용어
- 이해하고 기억할 핵심 정의

### 🎓 주요 교훈 (사례 기반)
- 저자의 현장 경험담
- 구체적인 문제 상황과 해결책
- "왜" 그렇게 하는지의 배경

### 💡 실행 가능한 가이드
- 단계별 실행 방법
- 실무에 바로 적용 가능한 기법
- 프로세스와 팁

### 📋 체크리스트
- 챕터 내용을 적용했는지 확인하는 항목
- 성능 문제 해결 시 참조할 체크포인트
- "yes/no"로 확인 가능한 질문 형식

### 🎯 한 문장 요약
- 챕터의 핵심 메시지를 한 문장으로
- 빠른 복습 시 참고
- 임팩트 있는 교훈 전달

---

## 🔍 챕터 그룹별 가이드

### 기초 이론 (00~02)
- **목표**: 성능 최적화의 기본과 방법론 이해
- **읽기 순서**: 반드시 이 순서대로
- **학습 시간**: 약 20분
- **핵심**: 관찰 → 질문 → 체계적 해결

### 측정 및 분석 (03~05)
- **목표**: 성능 문제를 측정하고 분석하는 법
- **핵심 스킬**: 프로파일링, 병목 식별, 예측
- **학습 시간**: 약 15분

### 최적화 전략 (06~12)
- **목표**: 성능 개선의 다양한 전략 학습
- **다루는 주제**: 지연, 낭비, 문제 해결, 예측, 병렬화, 테스트 등
- **학습 시간**: 약 30분

### 실무 관리 (13~15)
- **목표**: 계획, 조직, 커뮤니케이션
- **핵심**: 정치적 성공, 기술과 인간의 조화
- **학습 시간**: 약 10분
